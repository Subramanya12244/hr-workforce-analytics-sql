# HR Analytics SQL Project

## Project Overview

This project contains SQL solutions for common HR Analytics business problems using the IBM HR Employee Attrition Dataset.

The goal of this project is to strengthen SQL skills through real-world business scenarios involving:

- Employee Attrition Analysis
- Salary Analysis
- Department Analytics
- Job Role Analytics
- Scalar Subqueries
- Correlated Subqueries
- Aggregate Functions
- Conditional Aggregation

---

## Dataset

Dataset: IBM HR Analytics Employee Attrition Dataset

Key Columns:

- EmployeeNumber
- Age
- Gender
- Department
- JobRole
- MonthlyIncome
- Attrition
- YearsAtCompany
